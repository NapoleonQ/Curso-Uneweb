APIs para el desarrollo de aplicaiones de html5

A continuacion se muestran algunas de las interfaces para el desarrollo de aplicaciones empleando las APIs, que ofrece html5. Las cuales han sido estudiadas a lo largo del desarrollo de este curso

Definicion:

Corresponde a la utilidad practica, que provee la API, para mejorar las funcionalidades del sitio web.

Requerimientos

Son el conjunto de requisitos que deben encontrarse disponibles de a fin de que la API funcione en el navegador.

Parametros:

Corresponde a los argumentos de entrada, que debe resibir la API para producir el efecto requerido por el sitio web para su adecuado funcionamiento.

Resultados:

Son los efectos funcionales que se obtienen de la implementacion del componente web a implementar.
