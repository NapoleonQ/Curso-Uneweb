¿Qué es Arch Linux?

Arch Linux es una distribución de Linux independiente, optimizada para sistemas x86-64. Se basa en el principio KISS (Keep It Simple, Stupid), enfocándose en la elegancia del código y el minimalismo. Es un sistema de liberación continua o "rolling release", lo que significa que siempre tendrás el software más reciente.
